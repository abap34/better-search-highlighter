# Privacy Policy

Effective date: 2026-05-22

Better Search Highlighter processes page text and search terms locally in the user's browser to highlight exact matches on the current page.

## Data Collection

Better Search Highlighter does not collect, transmit, sell, or share:

- Page content
- Search terms
- Browsing history
- URLs visited by the user
- Extension settings

## Local Storage

The extension uses Chrome storage to save local preferences:

- Highlight level
- Blinking preference
- Search scope preference
- Search panel position

These settings are used only to provide the extension's user-facing functionality.

## Remote Code and Analytics

Better Search Highlighter does not use remote code, analytics, advertising, tracking pixels, or external servers.

## Permissions

The extension requests the narrow permissions needed for its search-highlighting feature:

- `activeTab`: run on the active tab after a user action
- `scripting`: inject the search panel and highlighter after a user action
- `storage`: save local preferences
- `contextMenus`: provide a right-click entry to open the search panel

## Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
