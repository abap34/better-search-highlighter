# プライバシーポリシー

施行日: 2026-05-22

Better Search Highlighter は、現在のページ上の完全一致検索と強調表示を行うために、ページ本文と検索語をユーザーのブラウザ内で処理します。

## データ収集

Better Search Highlighter は、以下の情報を収集、送信、販売、共有しません。

- ページ本文
- 検索語
- 閲覧履歴
- ユーザーが訪問した URL
- 拡張機能の設定

## ローカル保存

この拡張機能は Chrome storage を使い、以下の設定を保存します。

- 強調レベル
- 点滅の許可設定
- 検索範囲設定
- 検索パネルの位置

これらの設定は、拡張機能のユーザー向け機能を提供する目的でのみ使用されます。

## リモートコードと分析

Better Search Highlighter は、リモートコード、分析ツール、広告、トラッキングピクセル、外部サーバーを使用しません。

## 権限

この拡張機能は、検索ハイライト機能に必要な最小限の権限を要求します。

- `activeTab`: ユーザー操作後に現在のタブで動作するため
- `scripting`: ユーザー操作後に検索パネルとハイライト機能を注入するため
- `storage`: ローカル設定を保存するため
- `contextMenus`: 右クリックメニューから検索パネルを開くため

## Limited Use

Google API から受け取った情報の使用は、Limited Use 要件を含む Chrome Web Store User Data Policy に従います。
